Explicação da Problemática

Foi notado que a empresa a Cacau Show está recebendo muitas reclamações em relação à diversas áreas de produção e administrativas. Diante desse cenário, se faz necessário a elaboração de uma análise detalhada afim de criar planos de ações que servirão para evitar e minimizar os eventuais problemas gerados pelas reclamações recebidas. Sendo assim, os stakeholders apresentados a seguir foram acionados para liderar este projeto que irá implementar uma forma eficaz e funcinal de analisar, tratar, resolver e responder seus clientes, visando melhorar a satisfação e atração de novos compradores.

Métodos e processos das análises

Método/Processo:
Criação de um robô em Python

Detalhes

Foi desenvolvida uma solução em Python para a criação de um robô que é capaz de ler as informações dos comentários na página de reclamações do site Reclame Aqui e a biblioteca Pandas para realizar o tratamento dos dados sobre as reclamações.

Criação do robô

Método: Limpeza dos dados para melhor análise

Detalhes

Foi criado um script em Python para a filtragem e limpeza dos dados, utilizando diversas bibliotecas, dentre elas, a Pandas que é uma das principais bibliotecas utilizadas para a manipulação de dados. Também foi utilizado a ferramenta NLTK .

Limpeza dos dados

Método: Técnicas de agrupamento e visualização de dados

Detalhes

Foi desenvolvido um script em Python que através do algoritmo Kmeans , que é um algoritmo de clusterização. Ele funciona de forma não supervisionada (não precisa de inputs de confirmação) e avalia e agrupa os dados de acordo com suas características.

Soluções para os problemas encontrados

Problema 1

Problema: Maior quantidade de reclamações na Região Sudeste

Analisando o mapa de calor no primeiro gráfico, pode se notar que a região com maior alvo de reclamações é a região sudeste, mais específico no Estado de São Paulo.

Solução proposta

Problema 2

Problema: Maior numero de reclamações foi de produtos não recebidos.

Uma das maiores reclamações recebidas foi de produtos não recebidos. Já a menor foi de produto com defeito. O assunto de 'entregas' deve ser prioridade se tratando de uma loja online.

Solução proposta

Problema 3

Problema: Maior quantidade de reclamações ocorre em datas comemorativas

Analisando o gráfico temporal e fazendo um contraste com o grafico de tipo de reclamação, nota-se um problema na logisca e administração em datas onde a demanda cresce,

Solução proposta
